# Focus Mode

이 문서는 Tomado의 집중 모드가 사용자 관점에서 어떻게 동작해야 하는지 정리합니다.

구현 구조와 내부 책임은 [Timer Domain](../design-docs/timer.md)를 기준으로 확인합니다.

## 목표

- 사용자가 방해 요소를 줄인 전체 화면 환경에서 타이머에 집중할 수 있어야 한다.
- 메인 타이머 화면과 동일한 세션 상태를 집중 모드에서도 끊김 없이 확인할 수 있어야 한다.
- 배경 전환과 TODO 확인 같은 집중 모드 전용 상호작용이 자연스럽게 동작해야 한다.

## 핵심 사용자 흐름

### 1. 집중 모드 진입

1. 사용자는 메인 화면에서 집중 모드로 진입한다.
2. 전체 화면 오버레이가 열린다.
3. 현재 타이머 시간, 세션 인디케이터, 재생/일시정지/정지 버튼이 표시된다.
4. 처음 열릴 때 종료 방법을 안내하는 토스트가 한 번 표시된다.

### 2. 타이머 확인 및 제어

1. 사용자는 집중 모드 안에서도 현재 세션 시간을 본다.
2. 재생/일시정지 버튼으로 타이머를 제어할 수 있다.
3. 정지 버튼은 세션이 진행 중이거나 이미 시작된 경우에만 의미 있게 동작한다.
4. 메인 화면과 집중 모드는 같은 타이머 상태를 공유한다.

### 3. TODO 확인

1. 사용자는 좌측 패널에서 TODO 목록을 확인한다.
2. 기본적으로 TODO 영역은 접고 펼칠 수 있다.
3. 펼친 상태에서는 TodoPanel을 통해 입력/체크/삭제/정렬 흐름을 그대로 사용할 수 있다.

### 4. 배경 전환

1. 사용자는 좌우 배경 전환 버튼으로 배경을 바꿀 수 있다.
2. 방향키 좌우 입력으로도 같은 동작을 수행할 수 있다.
3. 마지막으로 선택한 배경은 다시 들어와도 유지된다.

### 5. 키보드 단축키

1. `Esc`로 집중 모드를 종료할 수 있다.
2. 방향키 `Left`, `Right`는 배경 이동을 수행한다.
3. 방향키 `Up`, `Down`은 TODO 패널 접기/펼치기에 사용된다.

### 6. 집중 모드 종료

1. 사용자는 우측 상단 버튼 또는 `Esc`로 집중 모드를 종료한다.
2. 종료해도 타이머 자체는 계속 유지된다.
3. 다시 진입하면 같은 세션 상태를 이어서 본다.

## 표시 규칙

- 집중 모드의 타이머 시간과 세션 인디케이터는 메인 화면과 동일한 기준을 사용한다.
- 집중 모드 배경은 시각적 몰입을 위한 전용 레이어로 표시된다.
- TODO 패널은 타이머보다 보조 정보로 취급되며 접기/펼치기가 가능해야 한다.

## 예외 및 실패 처리

- 집중 모드가 열려 있어도 타이머 상태는 별도로 유지되어야 한다.
- 배경 전환과 TODO 펼침 상태는 타이머 진행과 독립적으로 동작해야 한다.
- 집중 모드를 닫을 때 세션 상태나 현재 시간은 초기화되지 않아야 한다.

## 현재 구현 메모

- 집중 모드 전용 상호작용은 `useFocusModeController`가 담당한다.
- 타이머 표시 상태는 `AuthLayout`에서 공통 `timerSession`으로 조합한 값을 사용한다.
- 배경 index는 별도 persist store로 유지한다.

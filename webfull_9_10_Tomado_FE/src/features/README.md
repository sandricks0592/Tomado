# features

`features`는 서비스의 도메인 단위로 코드를 묶는 레이어다.

- 공용 UI가 아닌, 기능 의미가 있는 코드만 둔다
- 페이지는 조합만 하고, 도메인 상태와 동작은 `features`로 보낸다

상세 구조 원칙은 아래 문서를 기준으로 확인한다.

- [ARCHITECTURE.md](../../ARCHITECTURE.md)
- [FRONTEND.md](../../docs/FRONTEND.md)
- [Design Docs](../../docs/design-docs/index.md)

이 README는 `src/features` 레이어에서 문서 진입 경로를 빠르게 찾기 위한 안내 문서로 유지한다.
